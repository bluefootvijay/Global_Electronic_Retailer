Global Electronic Retailer: Sales Performance Analysis
This repository contains SQL scripts used to analyze the sales performance of a global electronic retailer, providing key insights into revenue trends, customer behavior, and regional contributions. The analysis is structured around the accompanying dashboard visualization () and a series of detailed sales metrics.

Analysis Overview
The following SQL queries dissect the retailer's sales data across multiple dimensions: time (month, year, quarter, day of the week), geography (store, country), and core financial metrics (revenue, profit, average order value).
